This program runs in Visual Studio 2015.

After running the program, all the user has to input is 1 - 2 based on their desired algortihm.

If the user enters 1, the program will execute and output the longest subsequence value.

If the user enters 2, the program will ask the user for a match score, a gap score, a similar score, and a dissimilar score. Then the program executes and outputs the optimal sequence alignment.
